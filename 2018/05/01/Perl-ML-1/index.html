Perl：Machine Learning

目前，大部分的机器学习框架均使用 Python、Java 等主流语言编写，Perl 在这一领域已经落后很多了。为了改善这种状况，Sergey Kolychev 用 Perl 编写了深度学习平台 MXNet 的 API 接口，并得到了 MXNet 官网的推荐。可从 MetaCPAN 网站下载 AI::MXNet。
