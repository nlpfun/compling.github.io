Hexo 简介

本网站使用 Hexo 创建。在制作过程中参阅了许多有关 Hexo 的网络资料，如 Hexo 官方文档、Hexo 常见问题汇总 和 Hexo GitHub 等。

Hexo 入门

Ubuntu 系统下安装和使用 Hexo 的步骤。

安装 Git

安装 Node.js

重启命令行界面后安装 Node.js

安装 Hexo

创建网站

新创建的网页文件保存在 ~/blog/source/_posts 文件夹，文件名为 hello-world.md。使用 Vim 打开该文件后可以修改文章标题，并添加标签。

多个标签可用逗号隔开。Hexo 支持 Markdown 语法，排版十分方便。

发布网站

网站发布后可使用 localhost:4000 预览网站内容。
